## Abstract

I'm an experimental physicist working on **memristor synthesis** and **neuromorphic devices** using polymers and ceramic materials, fabricating thin films via the sol-gel method. Simultaneously, I build high-performance desktop applications with **Rust** and **Tauri** &mdash; creating tools that bridge the gap between science and software.

From **Skardu, Gilgit-Baltistan** &mdash; where the mountains meet the sky.
