### About Me

I'm an experimental physicist pursuing my **MPhil in Physics** at **Quaid-e-Azam University, Islamabad**. I work on **memristor synthesis** and **neuromorphic devices** using polymers and ceramic materials, fabricating thin films via the sol-gel method.

When I'm not in the lab, I build desktop apps with **Rust/Tauri** and web apps with **Next.js**. I believe physics and code are two sides of the same coin.

From **Skardu, Gilgit-Baltistan** -- where the mountains meet the sky.
